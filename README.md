# 🎯 Real-Time Multiple Color Detection App

A real-time computer vision web application that detects **Red, Green, Blue, and Yellow objects** using OpenCV and displays bounding boxes with center tracking.

---

## 🚀 Live Demo

🔗 **Click here to try the app:**  
👉[ https://multiple-color-detection-app-xxxx.streamlit.app](https://multiple-color-detection-app-8zlxbdcry4tjjmqfsgnkva.streamlit.app/)

---

## 📌 Features

- 🎥 Real-time webcam detection
- 🎯 Center coordinate tracking
- 📦 Bounding box visualization
- 🌈 Multi-color detection (Red, Green, Blue, Yellow)
- ☁️ Deployed on Streamlit Cloud

---

## 🛠️ Tech Stack

- Python
- OpenCV
- NumPy
- Streamlit
- streamlit-webrtc
